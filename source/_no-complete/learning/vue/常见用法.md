## refs节点获取

```
// template
<button ref="myButton" @click="clickedButton">点击偶</button>

// script
let app = new Vue({ 
    methods: { 
        clickedButton: function () { 
            console.log(this.$refs.myButton); 
        } 
    } 
})
```

## Vue.nextTick

它可以在DOM更新完毕之后执行一个回调，用法如下：

```
// 修改数据

vm.msg = 'Hello'

// DOM 还没有更新

Vue.nextTick(function () {

// DOM 更新了

})
```
