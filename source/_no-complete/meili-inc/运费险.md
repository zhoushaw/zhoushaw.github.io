## 项目启动

> sdk基础安装

`meili sdk install`

> 依赖安装

`npm install`

> 初始化项目

`npm run init`

## 根目录下的router.json

后端路由分配到controll，controll对前端资源进行读取，每个page下又有前端独自的路由进行资源分配

webpack会将source打包到assets文件夹下
